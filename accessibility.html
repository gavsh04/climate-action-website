Accessibility Statement

We are committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to the Climate Action project.

Conformance Status

Standard Compliance

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. This website is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard (see Known Limitations).

Technical Features

Implemented Improvements

  • Skip to Content: We have implemented hidden "Skip to Content" links to allow keyboard users to bypass the navigation menu and jump directly to the main research articles.
  • Semantic Architecture: Use of <main>, <article>, and <aside> landmarks ensures screen readers can identify different areas of the page.
  • Interactive Components: The Gallery Lightbox and Interactive Simulator are built with ARIA roles (e.g., role="dialog" and role="progressbar") to communicate state changes to assistive technology.
  • Visual Clarity: We maintain a contrast ratio of at least 4.5:1 for all text. The primary dark green (#2E7D32) has been tested against white backgrounds for maximum legibility.
  • SVG Accessibility: The Sitemap uses <title> and <desc> tags within the SVG code to describe the site hierarchy to non-visual users.
Known Limitations

Current Challenges

Despite our best efforts, some areas of the site may have issues:

  • Simulator Complexity: The real-time updates in the Action Impact Simulator may be difficult for some older screen readers to track. We have added aria-live="polite" regions to mitigate this.
  • Third-party Media: Some photography sourced from Unsplash may contain complex visual data. We provide descriptive captions (.caption) to supplement the alt text.

Feedback

We welcome your feedback on the accessibility of the Climate Action website. Please let us know if you encounter accessibility barriers by contacting the Implementation Lead at the University of Westminster.
